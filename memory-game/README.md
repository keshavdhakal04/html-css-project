## 🧠 Memoji Game  
A fun and interactive memory card game where you flip, match, and win using animated emoji cards.

🔗 **Live Demo:** https://memoji.infinityfree.me/

---

## ✨ Features  
🃏 Flip and match emoji cards  
🎯 Dynamic grid customization (2×2 up to 8×8)  
📊 Real-time stats (Moves, Matches, Timer)  
⏱️ Auto-start timer on first move  
🎉 Win modal with performance summary  
🔄 Restart & Play Again functionality  
🎞️ Smooth flip, match, and shake animations  
📱 Fully responsive design
