# 🃏 Profile Card

A clean, animated personal profile card built with pure HTML & CSS.

🔗 **Live Demo:** [profile-card.infinityfree.me](https://profile-card.infinityfree.me/)

---

## ✨ Features

- 🎴 Animated fade-in card on load
- 🎨 Gradient header with **Open to Work** badge
- 🔵 Animated glowing profile image ring
- 🔗 Social links — GitHub, LinkedIn, Email
- 🏷️ Skill tags with hover effects
- 🔘 Action buttons with hover animations
- 📊 Stats section — Projects, Commits, Stars

---

## 🛠️ Built With

- HTML5
- CSS3
- [Boxicons](https://boxicons.com/)
- [Google Fonts - Poppins](https://fonts.google.com/specimen/Poppins)
