# ⏱️ Arc Ring Clock

A clean, animated clock that visualizes time using three circular SVG
progress rings.

🔗 **Live Demo:** https://arc-clock.infinityfree.me/

------------------------------------------------------------------------

## ✨ Features

-   ⭕ Three animated arc rings
-   🟡 Gold ring for **Hours**
-   🟣 Violet ring for **Minutes**
-   🔵 Cyan ring for **Seconds**
-   ⏰ Real-time **digital clock display**
-   🌙 **AM / PM indicator**
-   📅 Dynamic **date display**
-   📏 Auto-generated **SVG tick marks**
-   📱 Responsive design for mobile, tablet, and desktop
-   🎞️ Smooth ring animation using `stroke-dashoffset`

------------------------------------------------------------------------

## 🛠️ Built With

-   HTML5
-   CSS3
-   JavaScript (Vanilla JS)
-   SVG
